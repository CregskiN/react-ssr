1. csr 的前后端交互

> 前端 => ajax => 后端

> 后端 => json => 前端

2. 网页运行流程

   1. SSR

      - 发送请求&下载 html 文档
      - 页面就绪

   2. CSR
      - 发送请求&下载 html
      - 发送请求&下载 js(react) 文件
      - 浏览器运行 react 代码
      - 页面就绪

`TTFP` 首屏加载速度  
`SEO` 搜索引擎优化

| 技术类别 | TTFP |  SEO   |
| :------: | :--: | :----: |
|   CSR    |  短  |  支持  |
|   SSR    |  长  | 不支持 |
